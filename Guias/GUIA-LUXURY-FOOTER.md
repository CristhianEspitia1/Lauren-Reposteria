## 🚀 Integración en tus Páginas

### Paso 1: Agregar Google Fonts en `<head>`
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,600;1,600&family=Manrope:wght@400;500;600&display=swap" rel="stylesheet">
```

### Paso 2: Incluir CSS
```html
<link rel="stylesheet" href="../css/luxury-footer.css">
```

### Paso 3: Copiar Footer HTML
Abrir `_luxury-footer-snippet.html` y copiar todo el contenido al final del `<body>` de cada página.
